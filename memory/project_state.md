## What is complete (all committed and pushed)

### Core simulation
- Multi-modulation baseband: BPSK, DBPSK, MSK, QPSK, OQPSK, 8PSK, 16QAM, 16APSK, 32APSK
  (MSK built as offset-QPSK/half-sine — see technical_notes.md)
- N-carrier wideband simulation with shared NLA, per-carrier channel impairments, CNR/CIR/CNIR
- `sim/simulation.py`: the entry point is `simulate()` (renamed from
  `wideband_bpsk_simulation`, commit c60e6c3) and is built from named-phase
  helpers — `_composite_norm_factor`, `_prepare_carrier`, `_ChunkState`
  dataclass, `_process_chunk`, `_demod_carrier` (commit e192622)
- OLA upsample/downsample, chunk pipeline (O(1) wideband RAM), chunk-progress callbacks
- 2D IBO × noise density sweep with adaptive Wilson-CI iteration per point
  (num_symbols / num_frames derived from a memory budget; see technical_notes.md
  § "Adaptive iteration")
- **Either-or convergence:** optional `[simulation].target_ci_relative` (e.g. 0.01 =
  ±1% of BER) — a sweep point stops when EITHER the absolute or the relative
  half-width target is met (commit 520cee0)
- FEC coding: convolutional (K=7, rate-1/2, soft Viterbi), concatenated (RS+convolutional),
  turbo (rate-1/3 PCCC, max-log-MAP BCJR), LDPC (normalized min-sum BP)
  — wired into the TX/RX chain via [carrier.coding]

### Phase noise (per-carrier oscillator model)
- `sim/phase_noise.py`: `(offset_hz, dbc_per_hz)` mask, log-log interpolated, flat
  extrapolation; generated by frequency-domain coloring; applied as `exp(j·φ[n])`
  at each carrier's native rate, right after the channel filter
- Config is per-carrier: `[carrier.phase_noise]` with `enabled`, `offset_hz`,
  `dbc_per_hz` (commit fe7c1fe; originally global, moved per-carrier same day)
- Shows up in BER/EVM but NOT CIR/CNIR (projection treats it as part of the
  signal — documented tradeoff in technical_notes.md)

### Implementation loss for APSK (no closed-form BER)
- `tools/generate_theory_curves.py` generates BER-vs-Eb/N0 reference curves with
  the **direct-AWGN chain** (matches tests/test_awgn_performance.py — no OLA, no NLA)
- Committed tables: `data/theory/ber_awgn_16APSK.npz` (BER 0.19→1.9e-7),
  `ber_awgn_32APSK.npz` (BER 0.19→4.1e-7) (commit f2a6d93)
- `sim/theory.py` loads them so `ber_awgn` / `ebn0_for_ber` work for 16APSK/32APSK;
  IL is now reported for APSK carriers in report.md.  Non-default gammas suppress
  the lookup (returns None).  See technical_notes.md § "APSK reference tables"
- NOTE: an earlier full-chain generator was abandoned because an identity AM-AM
  table clipped high-PAPR APSK peaks and faked 4–13 dB of "chain loss"

### GUI — schema-driven, with reusable tkconfig package
- `tkconfig/` (NEW package, commit dc26b2d) — project-agnostic schema-driven tkinter
  form toolkit:
  - `schema.py`   Field / Section / Tab dataclasses (Field supports types int,
                  float, float_optional, str, path, str_enum, bool, float_list,
                  float_list_text; plus `options` for combobox and `visible_when`
                  for conditional visibility)
  - `widgets.py`  Tip, scrollable, labeled, entry, fmt, parse_float_list, make_browse_cb
  - `serde.py`    walk_fields, cfg_get/cfg_set, populate_from_schema, collect_from_schema
  - `render.py`   render_tab / render_section (+ private scalar/field/visibility helpers)
  - `toml_writer.py`  lit / arr / emit_table
- `gui.py` now holds only project-specific code: the Field/Section/Tab schemas
  (`_GENERAL_TAB`, `_SWEEP_OUTPUT_TAB`, `_AMPLIFIER_TAB`, and carrier sub-schemas),
  the App, CarrierFrame, and build_toml.  Dropped from a ~1380-line peak to ~941.
- Every tab is built from a schema via render_tab / populate_from_schema /
  collect_from_schema.  FEC's LDPC-matrix field uses `visible_when=(("scheme",),
  ("ldpc",))` so it shows only for ldpc.
- Carrier sub-sections (Channel / Phase noise / FEC) collapse via grid_remove()
  when unchecked and cache their values so toggling off/on restores them (commit 0028edd)
- App icon generated at runtime (misc.gen_icon.build_icon()); stdout teed to
  `simulation.log`; live progress bar + log pane

### Progress feedback during runs

## Output files (fixed names)

Written to `[output].output_dir`. `[output].plots` (default true) gates images;
`report.md` is always written when any carrier has `sweep_demod = true`.

- `wideband.png` — composite wideband PSD
- `amplifier.png` — AM-AM / AM-PM table plot
- `<name>_detector.png` — per-carrier 2×3 BER/EVM/CNR-family grid
- `<name>_detector_<panel>.png` — each of the six detector panels as a standalone
  PNG (panels: ber_vs_ibo, evm_vs_ibo, db_vs_ibo, ber_vs_cnr, evm_vs_cnr,
  db_vs_cnr) (commit 8848d4e)
- `<name>_channel.png` — per-carrier channel impairment response
- `<name>_phase_noise.png` — per-carrier phase-noise mask + cumulative RMS jitter
- `report.md` — flat (carrier, IBO, noise) results table; includes APSK IL

`<name>` is the carrier name with spaces replaced by underscores.
